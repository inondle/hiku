# hiku

A haiku-based chat app.

App is an Expressjs web server with a Knockoutjs based frontend. 

## Run Locally with Docker
 * install docker 
 * clone the repo
 * `touch src/db/dev.db`
 * `docker-compose up`
 * when done: `docker-compose down`

## Run Locally
Steps to get app running locally:

 * clone the repo
 * install node
 * install sqlite
 * `npm install`
 * `touch src/db/dev.db`
 * `node ./src/server/server.js`
